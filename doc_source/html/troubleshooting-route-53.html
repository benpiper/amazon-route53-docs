Troubleshooting Amazon Route 53

The topics in this chapter can help you troubleshoot problems with your domain registration and DNS configuration.

Topics + My domain is unavailable on the internet + My domain is suspended (status is ClientHold) + Transferring my domain to Amazon Route 53 failed + I changed DNS settings, but they haven’t taken effect + My browser displays a “Server not found” error + I can’t route traffic to an Amazon S3 bucket that’s configured for website hosting + I was billed twice for the same hosted zone + My AWS account is closed, suspended, or terminated, and my domain is registered with Route 53
